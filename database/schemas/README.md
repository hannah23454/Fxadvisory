# MongoDB Schemas

This folder documents the MongoDB collections used in SwitchYard FX.
TypeScript interfaces live in `lib/types/models.ts`.

## Collections

| Collection    | Purpose                              |
|---------------|--------------------------------------|
| users         | Authenticated user accounts          |
| sessions      | NextAuth session tokens              |
| preferences   | Per-user feed & notification prefs   |
| bookings      | Meeting / calendar bookings          |
| messages      | In-app messaging threads             |
| polls         | Weekly insight poll votes            |
| trades        | FX trade records                     |
